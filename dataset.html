Data Format

Each dataset is provided as a CSV file with the following columns:

smiles,sequence,stereo_label,property_1,property_2,...,split
CC[C@@H](N)C(=O)...,ACDEF,L,0.82,1.45,...,train
CC[C@H](N)C(=O)...,ACDEF,D,0.21,1.45,...,train
  • smiles — Canonical SMILES string encoding full stereochemistry
  • sequence — Amino acid sequence (one-letter code)
  • stereo_label — L (natural) or D (non-natural) chirality
  • property_* — Experimental property values (NaN if not measured)
  • split — Assigned split: train / valid / test
